LSTM for Sales Forecasting

Brief Overview:

This project is aimed at aiding retail stores accurately forecast their weekly sales by utilizing the trained multivariate sales forecast model (time series forecast model) to predict the sales. A Long Short Term Memory (LSTM) Neural Network model has been trained to achieve this. Simply by providing 11 predictor attribute values of past 11 weeks as input to this system, the sales for the upcoming 12th week can be forecasted. To view the functionality of this application, the example data can be downloaded using the link provided below in the instructions section and following the same instructions.

    Data Description:

    The Time Series Dataset contains 3 CSV files (‘features.csv’, ‘stores.csv’ & ‘train.csv’) that have all the predictor attributes and the target attribute (‘weekly sales’) data available across the 3 files. This dataset contains ~421K rows of weekly sales for 99 departments, and 45 stores of 3 store types, across 143 weeks.
    The ‘train.csv’ is a historical dataset which covers weekly sales of every store & department between 2010-02-05 to 2012-11-01. It contains the attributes, ‘store number’, ‘department number’, ‘weekly sales’, ‘IsHoliday’, and ‘Date’.
    The stores.csv file contains anonymized information about the 45 stores, indicating the type and size of store.
    The features.csv file contains additional data related to the store, and regional activity for the given dates such as ‘Temperature’, ‘Fuel Price’, ‘Consumer Price Index’ and ‘Unemployment Rate’.
